# Matthews Fitness – Account & Data Deletion

Users of the Matthews Fitness mobile application may request deletion of their account and all associated data at any time.

## How to request account deletion

### Option 1: In-App (Recommended)
1. Open Matthews Fitness
2. Go to **Settings → About Us → Delete Account**
3. This will generate a pre-filled deletion request email

### Option 2: By Email
1. Send an email to **matthewsfitnesshelp@gmail.com**
2. Use the subject line: **Account Deletion Request**
3. Include **your Matthews Fitness Account ID (Support ID)**, which can be found in the app under **Settings → About Us**
4. If available, also include the email address associated with your Matthews Fitness account

Providing the Account ID helps us accurately locate and delete the correct account.

## Data that will be deleted

Upon request, the following data associated with the account will be permanently deleted:

- User account and profile information
- Nutrition and food logs (including barcode scans, manual entries, and AI-assisted entries)
- Weight and body composition history
- Workout plans and activity data
- AI-generated insights, coaching evaluations, and recipes linked to the account
- User preferences, settings, and subscription status
- Any photos or images previously submitted for AI analysis

## Data retention

- Deletion requests are processed within **30 days**
- Certain data may be retained temporarily for legal, security, or operational purposes as required by law
- Matthews Fitness does **not** sell user data

For questions related to data or account deletion, contact **matthewsfitnesshelp@gmail.com**.
